# Simple Website

This project is a simple HTML website that displays a list of current projects. It is designed to be lightweight and easy to understand, making it suitable for beginners who want to learn about web development.

## Project Structure

```
simple-website
├── index.html       # Main HTML document
├── css
│   └── styles.css   # Styles for the website
├── js
│   └── scripts.js    # JavaScript for dynamic content
└── README.md        # Documentation for the project
```

## Getting Started

To set up this project locally, follow these steps:

1. Clone the repository or download the project files.
2. Open the `index.html` file in your web browser to view the website.
3. You can modify the `css/styles.css` file to change the appearance of the website.
4. Update the `js/scripts.js` file to manage the list of current projects dynamically.

## Features

- Displays a list of current projects.
- Simple and clean design.
- Easy to customize and extend.
